10 lines to a running agent
from tename import Tename

client = Tename()

agent = client.agents.create(
    model="claude-opus-4-6",
    system_prompt="You are a research assistant.",
    tools=["web_search", "python"],
)

session = client.sessions.create(agent_id=agent.id)

for event in session.send("Research the EV charging market"):
    if event.type == "assistant_message":
        print(event.payload["content"], end="")

Switch to GPT-5? Change model="gpt-5" — one line, same code.

Durable sessions

Append-only event log with idempotent writes. Kill the process mid-run, restart, resume exactly where you left off. Zero lost work.

Per-model profiles

YAML configs that tune caching, context management, and quirk handling per model. Same harness, different optimization. Community-contributed.

Sandbox isolation

LLM-generated code runs in Docker containers. Credentials never enter the sandbox — vault + tool proxy keeps secrets structurally separate.

Framework-agnostic

Tename doesn't replace Deep Agents or LangGraph — it runs beneath them. Bring your existing code, get production infrastructure underneath.

Where Tename fits

Tename is the runtime layer. Your framework handles the thinking. Tename handles everything else.
